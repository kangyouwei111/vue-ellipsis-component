# 介绍

`vue-ellipsis-component` 是一个高性能、可自定义的 vue 文本缩略组件。支持自定义缩略符、尾文本过滤、富文本缩略等功能。

灵感来自于 [react-ellipsis-component](https://github.com/chenquincy/react-ellipsis)。

# 功能

- 文本内容缩略
- 富文本内容缩略
- 自定义缩略符号
- 尾文本过滤
- 支持超过 n 行展示 m 行
- 支持缩略回调事件
